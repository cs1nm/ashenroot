# Ashen Roots

Ashen Roots is a 2D pixel sandbox adventure inspired by the structure of Terraria, but built around a different fantasy: the world remembers a destroyed civilization through ash, roots, ruins, and underground biomes.

## First Prototype Goals

- Seed-based world generation.
- Surface, caves, ores, trees, and small ruins.
- Player movement with jumping.
- Mine and place blocks.
- Basic HUD with seed and controls.
- Class data prepared for later item and armor systems.

## Core Loop

1. Explore the surface and caves.
2. Mine blocks and ores.
3. Build a safe base.
4. Craft class gear.
5. Find biome structures.
6. Defeat bosses to unlock deeper world states.

## Planned Classes

- Warrior: melee weapons and rage.
- Shieldbearer: block, guard, shockwaves.
- Archer: bows and special arrows.
- Sniper: firearms, charged critical shots.
- Mage: mana, elements, control.
- Summoner: minions and biome traits.
- Alchemist: flasks, bombs, reactions.
- Engineer: turrets, traps, devices.
- Memory Reaper: ash magic and low-health power.

## Next Milestone

The next milestone should add an inventory, item drops, tool strength, and a small crafting bench. After that, enemies and the first cave boss can be added without rewriting the world core.
